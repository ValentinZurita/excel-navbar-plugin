# MVP Definition

## MVP Goal

Deliver a first usable Excel add-in that makes worksheet discovery and organization meaningfully better than Excel’s native tab bar for workbooks with 20–80 sheets.

## What IS MVP

- Ribbon entry point + task pane experience
- Sidebar-based worksheet navigator
- Search by sheet name
- Manual groups
- Group management
- Pinning
- Hidden sheets section
- Immediate navigation on selection
- Excel-aligned minimal UI

## What is NOT MVP

- Smart suggestions
- Recent usage intelligence
- Automatic grouping
- Advanced ranking logic
- Reordering actual Excel worksheet order
- Preview-first navigation flows

## Risks and Assumptions

- Users are willing to manage groups manually if the workflow is fast and simple.
- Search by sheet name is enough for the first MVP.
- A task pane can feel sufficiently integrated if the UI is carefully designed.
