Sheet Navigator

Modern worksheet navigation for Excel

What it does

Sheet Navigator replaces Excel's built-in sheet tab bar with a fast, organized task pane. Group related worksheets, search by name, reorder with drag & drop, and navigate entirely from the keyboard.

Groups

Organize sheets into collapsible groups to reduce clutter.

Search

Jump to any worksheet instantly with live filtering.

Drag & Drop

Reorder sheets and move them between groups visually.

Keyboard-first

Navigate, rename, hide, and switch sheets without a mouse.

Install

  1. Download the manifest.xml file.
  2. In Excel, go to Insert → Add-ins → Upload My Add-in.
  3. Select the downloaded manifest.xml and confirm.
  4. The Sheet Navigator button appears in the Home tab.
